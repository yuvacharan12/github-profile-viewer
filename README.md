# 🚀 GitHub Profile Viewer

A simple and interactive web application that allows users to search for any GitHub profile and view their details along with top repositories.

---

## 📌 Features

* 🔍 Search any GitHub username
* 👤 View profile details (name, bio, followers, following)
* 📦 Display public repositories
* 🔗 Clickable repository links
* ⚡ Real-time data using GitHub API
* ❌ Error handling for invalid usernames
* ⏳ Loading state for better user experience

---

## 🛠️ Tech Stack

* HTML
* CSS
* JavaScript (ES6)
* Fetch API
* DOM Manipulation
* Async/Await

## ⚙️ How It Works

1. User enters a GitHub username
2. App sends request to GitHub API
3. Fetches profile data and repositories
4. Displays data dynamically on UI

---

## 🧠 Key Concepts Used

* API Integration using Fetch
* Dynamic DOM Manipulation
* Error Handling
* Conditional Rendering
